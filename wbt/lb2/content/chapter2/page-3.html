Interquartilsabstand

Der Interquartilsabstand kann mit der Funktion IQR() ausgegeben werden:

IQR(
  pss$stfdem, 
  na.rm = TRUE
)

{{% expand "Wie interpretierst du diese Zahl?" %}} Der Abstand zwischen dem ersten und dritten Quartil beträgt 3! Dieser Wert sagt uns noch nicht, wo das erste und das dritte Quartil liegen.

\[q_{75} - q_{25} = IQR\] {{% /expand %}}

Alternative Ausgabe des IQR

Eine etwas anpassungsfähigere Alternative ist die Kombination der Funktionen diff() und quantile(). In der Funktionquantile() wird die Variable angegeben (\(1.\) Argument), die gewünschten Quantile (\(2.\) Argument) und zuletzt werden NA's ausgeschlossen. Die Funktion liefert zwei Werte und mit der Funktion diff() wird von diesen zwei Werten dann die Differenz berechnet und ausgegeben.

diff(
  quantile(
    pss$stfdem, 
    probs = c(
      0.25,
      0.75
    ),  
    na.rm = TRUE
  )
)

Standardabweichung & Varianz

Die Standardabweichung und die Varianz erhalten wir über die folgenden zwei Funktionen:

sd(
  pss$stfdem, 
  na.rm = TRUE
)

var(
  pss$stfdem, 
  na.rm = TRUE
)

{{% expand "Wie könntest du alternativ von der Standardabweichung auf die Varianz kommen?" %}} Du weißt, dass die Standardabweichung nur die Wurzel der Varianz ist. Um also von der Standardabweichung auf die Varianz zu kommen, muss diese einfach quadriert werden. Das kann als einfache Rechnung umgesetzt werden:

sd(pss$stfdem, na.rm = TRUE) ^ 2

{{% /expand %}}
